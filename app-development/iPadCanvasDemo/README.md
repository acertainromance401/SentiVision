# iPadCanvasDemo

SentiVision iPad demo app source.

## Folder Layout

- `iPadCanvasDemo.xcodeproj` - Xcode project
- `iPadCanvasDemo/iPadCanvasDemo/App` - app source files
- `iPadCanvasDemo/iPadCanvasDemo/Data` - bundled CSV data
- `iPadCanvasDemo/iPadCanvasDemo/Info.plist` - app configuration

## Open in Xcode

1. Open `app-development/iPadCanvasDemo/iPadCanvasDemo.xcodeproj` in Xcode.
2. Select the `iPadCanvasDemo` scheme.
3. Choose an iPad Simulator.
4. Run with `⌘R`.

## What is inside

- Initial setup screen for profile name, baseline emotion, baseline color, and perception strength
- Personal RGB 3D distribution view with point details and camera reset
- PencilKit canvas with pen, eraser, color palette, and thickness slider
